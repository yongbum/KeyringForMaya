# KeyringForMaya
 Keyring회사에서 사용하는 마야 스크립트


# 파일 명 컬러
 - 초록색 : 새로 추가된 파일
 - 파랑색 : 수정된 파일
 - 빨간색 : 삭제된 파일

# commit -> VCS / Git / Commit FIle...

# clone : 저장소에 저장된 프로젝트를 복사합니다.
# add : 작업한 파일을 커밋하기위해 추가합니다.
# commit : 로컬 저장소에 저장합니다.
# push : 서버 저장소에 로컬 저장소를 업로드합니다.

# pull : 서버에 있는 데이터를 불러옵니다.

# merge : 충돌 난 파일을 합치는 과정.
